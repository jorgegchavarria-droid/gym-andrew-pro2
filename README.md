# GymRoutine Pro - Aplicación Completa de Entrenamiento

Una aplicación web completa para principiantes que quieren empezar en el gimnasio desde cero. Incluye rutinas de entrenamiento, biblioteca de ejercicios, seguimiento de progreso y mucho más.

## 🏋️‍♂️ Características Principales

### 📋 Rutinas de Entrenamiento
- **Rutina Full Body Principiante**: Perfecta para empezar, 3 días por semana
- **Rutina Empuje/Tirón/Pierna**: División clásica para nivel intermedio
- **Rutina Upper/Lower Split**: División superior/inferior para mayor frecuencia
- Todas las rutinas incluyen series, repeticiones e instrucciones detalladas

### 🏃‍♂️ Biblioteca de Ejercicios
- **26 ejercicios completos** cubriendo todos los grupos musculares:
  - Pecho (4 ejercicios)
  - Espalda (4 ejercicios) 
  - Piernas (5 ejercicios)
  - Hombros (4 ejercicios)
  - Brazos (4 ejercicios)
  - Abdomen (5 ejercicios)
- Instrucciones detalladas para cada ejercicio
- Sistema de filtrado por grupo muscular
- Búsqueda de ejercicios
- Información de calorías y dificultad

### 📊 Seguimiento de Progreso
- Estadísticas semanales de entrenamiento
- Contador de calorías quemadas
- Racha de días consecutivos
- Historial de entrenamientos
- Espacio para fotos de progreso

### ⏱️ Temporizador Integrado
- Temporizador de descanso entre ejercicios
- Contador de series y repeticiones
- Interfaz clara durante el entrenamiento

### 💾 Almacenamiento Local
- Todos tus datos se guardan localmente
- Tu progreso se mantiene entre sesiones
- Sin necesidad de registro o conexión

## 🚀 Cómo Empezar

### Requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- No requiere instalación adicional

### Instrucciones
1. **Descarga los archivos** de la aplicación
2. **Abre `index.html`** en tu navegador web
3. **Explora las diferentes secciones**:
   - **Dashboard**: Resumen de tu progreso
   - **Rutinas**: Elige tu plan de entrenamiento
   - **Ejercicios**: Aprende la técnica correcta
   - **Progreso**: Seguimiento de tus avances

### Uso Rápido
1. En el dashboard, haz clic en "Empezar Mi Entrenamiento"
2. Selecciona una rutina según tu nivel
3. Sigue las instrucciones ejercicio por ejercicio
4. Usa el temporizador de descanso entre series
5. Tu progreso se guardará automáticamente

## 💡 Consejos para Principiantes

### Antes de Empezar
- **Consulta con un médico** antes de iniciar cualquier rutina de ejercicios
- **Calienta siempre** 5-10 minutos antes de entrenar
- **Estira al final** para mejorar la flexibilidad

### Durante el Entrenamiento
- **Concéntrate en la técnica** antes que en el peso
- **Respira correctamente**: exhala en el esfuerzo, inhala en la relajación
- **Descansa entre series** 60-90 segundos para principiantes
- **Mantén la hidratación** durante todo el entrenamiento

### Progresión
- **Aumenta gradualmente** el peso o las repeticiones
- **Escucha a tu cuerpo** y descansa cuando sea necesario
- **Sé constante** - la regularidad es más importante que la intensidad
